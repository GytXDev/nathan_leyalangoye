Portfolio professionnel de LEYALANGOYE LEBOGHO Japhet Nathan, développeur logiciel spécialisé en développement web & mobile, systèmes métier complexes et intégration ERP Odoo.

## Technologies utilisées

- [Next.js](https://nextjs.org) 16.1.1
- [React](https://react.dev) 19.2.3
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS](https://tailwindcss.com) 4
- [Framer Motion](https://www.framer.com/motion) pour les animations

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
npm start
```
